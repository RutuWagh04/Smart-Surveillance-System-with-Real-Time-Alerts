# Smart Surveillance System with Real-Time Alerts

## Introduction

The **Smart Surveillance System with Real-Time Alerts** is a real-time security monitoring solution designed to detect suspicious activities and immediately notify users. The system focuses on live video monitoring and intelligent event detection without relying on database storage, making it lightweight and efficient.

It is suitable for homes, offices, campuses, and public areas where quick response and continuous surveillance are required.

---

## Key Features

### Admin Module

* Camera Configuration & Management
* Live Surveillance Dashboard
* Alert Control & Monitoring
* System Settings & Access Control
* Activity Log Monitoring (Real-Time)

---

### Surveillance Module

* Live Video Streaming
* Motion Detection
* Intrusion Detection
* Face / Object Detection (Optional)
* Real-Time Video Processing

---

### Alert & Notification Module

* Instant Alerts on Suspicious Activity
* Email / SMS / App Notifications
* Emergency Alert Triggering
* Real-Time Event Display

---

### User Module

* Secure Login System
* Live Camera Access
* Alert Viewing in Real Time
* Remote Monitoring Capability

---

## Highlights

* **No Database Dependency:** Operates without storing data in a database.
* **Real-Time Alerts:** Immediate notifications for detected threats.
* **Lightweight System:** Faster performance with minimal resource usage.
* **Remote Access:** Monitor live feeds from anywhere.
* **Scalable Architecture:** Easy integration of additional cameras or sensors.

## Technologies Used

* **Front-End:** HTML, CSS, JavaScript

* **Computer Vision:** OpenCV (for motion or object detection)
* **Alert System:** Email / SMS / Push Notifications
* **Hardware (Optional):** CCTV / IP Cameras, Sensors
